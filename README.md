# SigmaDSP-Marshall-BT-Speaker
DIY Marshall Middleton-style portable Bluetooth speaker using  ADAU1701 SigmaDSP, ESP32-WROVER with LDAC/aptX/AAC support,  TPA3116 + TPA3110 Class-D amps, and live EQ tuning via I²C.  CY7C68013A used as DIY USBi programmer. Currently in design  phase — hardware build coming soon.
